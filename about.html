About this Site

1-Shot PC was built using standard web technologies by me, Rylee Corradini. It's a Progressive Web App, meaning you can install it directly to your tablet/phone screen/desktop like a regular app, and it will behave more or less the same as one, and save its data in your device storage - no login needed.

It's made for one very simple purpose: creating random 1st-level characters using the D&D 2024 ruleset, for running one-shot game nights. If you want something that lets you build higher-level characters, multi-class, have more control over the build... well, there are plenty of other tools out there for that. :^)

Most art assets are from SVG Repo. The cool "graph paper" background effect is a hacked version of something originally done by Lea Verou (who is an awesome person to follow in web dev, by the way). All were tweaked by me, because open standards like CSS and SVG are rad that way.
